# Jetbrains-Enhancement-tool
Jetbrains系列产品增强工具

# 激活方式说明

#### 方式1：无限重置30天试用期补丁

适合用于2021.3及以下版本

#### 方式2：永久激活补丁

适合用于2022.1及以上版本

#### reset_script

适合用于目前已经过期的机器




# 支持的产品
- IntelliJ IDEA
- AppCode
- CLion
- DataGrip
- GoLand
- PhpStorm
- PyCharm
- Rider
- RubyMine
- WebStorm
